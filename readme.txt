== Description ==

You need to embed multiples youtube video? Then this plugin is for you!
This plugin is used to display video in iframe using youtube watch url. This is boon if want to add more youtube video directly on your post or pages.

= Features =

1. There is a setting option where can configure video iframe display height & width.
2. Option to add more than one video according to your needs.
3. Edit, Update & Delete any time.
4. No compulsion to add video on each page.
5. Easy to display by using shortcode on page or post.

== Installation ==

1. Unzip and upload the `best-youtube-video` directory to the plugin directory ('/wp-content/plugins/')
2. Activate the plugin through the 'Plugins' menu in WordPress
3. And then in admin menu a "Youtube Video Setting" menu appear.
4. Open it and se width & height of iframe.
